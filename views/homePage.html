🚀 Agile Demo Project Overview

This project simulates the development of an online beverage store through three Agile iterations. The goal is not to create a complete commercial system, but to demonstrate:

  • ✔️ Understanding of Agile development
  • ✔️ Hands-on use of Docker for containerization
  • ✔️ End-to-end cloud deployment via AWS EC2

⚙️ Cycle 1 – Initial Client Delivery

This version contains only two static HTML pages (Home & Product) with no server-side functionality.

Cycle 1 - Home Page Cycle 1 - Product Page

⚙️ Cycle 2 – Added Basic Interactivity

Refined UI + Express server with basic page routing.

Cycle 2 - Entry Page

✅ Cycle 3 – Final Delivery

Node.js backend + MySQL integration + Docker deployment to AWS EC2.

Cycle 3 - Entry Page

📦 Key Highlights

  • DevOps Flow: Code → Container → Deploy
  • Agile Iteration: Rapid delivery in 3 client-visible stages
  • Cloud-Ready: Real deployment via AWS EC2
  • End-to-End Stack: Frontend, backend, DB, and CI thinking

"Build fast, learn faster, deploy like it's real."
